 Flutter Clothing eCommerce App

A modern mobile eCommerce application built using Flutter.  
This app allows users to browse clothing items, create accounts, and shop seamlessly.

---

 Features

- 🔐 User Authentication (Login & Sign Up)
- 👤 Continue as Guest option
- 🏠 Home Screen
- 🛍️ Product browsing (coming soon)
- 🛒 Cart system (in progress)
- 💳 Checkout system (planned)



🖼️ Screens

- Login Page  
- Signup Page  
- Home Screen  

 Tech Stack

- Frontend: Flutter (Dart)
- State Management: GetX
- Local Storage: GetStorage
- Backend: (Planned - PHP / Firebase)
- Payments: (Planned - M-Pesa Integration)
